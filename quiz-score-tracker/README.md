# Quiz Score Tracker

This project is a simple web application that tracks the scores of a quiz with four participants. It provides a user-friendly interface to display participant names and their respective scores, allowing for easy updates and management of the quiz scores.

## Project Structure

```
quiz-score-tracker
├── src
│   ├── index.html       # Main HTML document for the quiz score tracker
│   ├── style.css        # Styles for the webpage
│   └── app.js           # JavaScript code for score tracking functionality
├── package.json         # Configuration file for npm
└── README.md            # Documentation for the project
```

## Getting Started

To get started with the quiz score tracker, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd quiz-score-tracker
   ```

3. **Install dependencies**:
   ```
   npm install
   ```

4. **Open the `index.html` file in your web browser** to view the quiz score tracker.

## Usage

- Enter the names of the participants in the designated fields.
- Update scores as the quiz progresses.
- The current scores will be displayed in real-time.
